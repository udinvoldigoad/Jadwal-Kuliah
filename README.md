# Jadwal Kuliah

Aplikasi web sederhana untuk mengelola **jadwal mata kuliah** dan (opsional) **daftar tugas** perkuliahan.

## Fitur Utama
- ✅ CRUD Jadwal Mata Kuliah (tambah, ubah, hapus)
- ✅ Tampilan jadwal per hari
- ✅ Detail jadwal: jam, nama mata kuliah, SKS, kelas, ruangan, dosen
- ✅ (Opsional) Menu Tugas: tambah tugas, tandai selesai, pisah Upcoming & Completed
- ✅ Pencarian (search) untuk jadwal/tugas
- ✅ UI compact & mobile-friendly

## Tech Stack
- **React** + **Vite**
- **Tailwind CSS**
- (Opsional) **Local Storage / Database** untuk penyimpanan data
